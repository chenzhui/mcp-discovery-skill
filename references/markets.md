# MCP Markets

Start with these public MCP discovery surfaces:

1. `https://www.mcpworld.com`
   - Broad public catalog
   - Good for category browsing and quick Chinese-language discovery
2. `https://smithery.ai`
   - Well-known MCP registry and install surface
   - Good when searching by integration or package name
3. `https://glama.ai/mcp`
   - Broad MCP catalog with public landing pages
   - Useful as a secondary cross-check

Treat this list as a starting priority list, not an absolute authority ranking. If search quality or maintenance signals are weak, fall back to direct GitHub search.

## Search Hints

- Search the market first for product-facing names.
- Search GitHub next for implementation detail, stars, commits, and install docs.
- When looking for a skill rather than an MCP, prefer GitHub and local skill repositories.
